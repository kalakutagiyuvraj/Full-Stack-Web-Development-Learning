CSS display Property - Theory

The display property controls how an element is shown in the browser. It defines the element’s layout behavior (block, inline, grid, flex, etc.).

Common Values

  • block → Takes full width, starts on new line. Examples: <div>, <p>.
  • inline → Takes only needed width, does not break line. Examples: <span>, <a>.
  • inline-block → Behaves like inline but allows width/height.
  • none → Hides element completely (not in layout).
  • flex → Enables Flexbox layout for arranging items.
  • grid → Enables Grid layout for rows/columns.

Other Values

  • inline-flex → Flexbox but inline-level.
  • inline-grid → Grid but inline-level.
  • table, table-row, table-cell → Behaves like table elements.
  • list-item → Behaves like a list item (<li>).
Tip: display only controls how the element is shown, not its position (use position, margin, padding for that).
