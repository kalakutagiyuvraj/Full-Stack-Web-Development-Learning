CSS stands for Cascading Style Sheets.

It is a stylesheet language used to control the look and layout of web pages written in HTML.

HTML structures the content, and CSS styles it (colors, fonts, spacing, positioning, animations, etc.).

Element	p {}	Targets all <p> tags
