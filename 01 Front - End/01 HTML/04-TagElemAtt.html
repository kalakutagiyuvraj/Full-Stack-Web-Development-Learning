Tags - Elements - Attributes

        1. Tags : 
        a. Tags are the building block of HTML. They are used to create element structures in a web page.
        b. Tags are enclosed in angle brackets, eg <tagname>. These are commonly comes in pairs, with an opening and closing tag.
    
        2. Elements : 
        a. An element is complete structure of a tag, including its content and attributes.
        b. An element consists of an opening tag, content, and a closing tag, eg <tagname>content</tagname>.
    
        3. Attributes : 
        a. Attributes provide additional information about an element.
        b. They are specified within the opening tag and consist of a name and a value, eg <tagname attrbute="value">.
    
        4. Example : 
        <h1 class="title" id="main-heading">Hello, World!</h1>
        a. In this eg <h1> is the tag, "Hello, World!" is the content, and class and id are the attributes.
        b. The class attribute is used to apply CSS styles on a particular group of elements, while the id used for unique element.
