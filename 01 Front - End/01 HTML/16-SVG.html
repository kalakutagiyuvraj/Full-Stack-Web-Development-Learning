âœ… What is SVG?
SVG stands for Scalable Vector Graphics.
It is an XML-based format for creating vector images directly in HTML.

Unlike normal images (like .jpg or .png), SVG images do not lose quality when zoomed or resized.

ðŸ§  Why Use SVG?
Feature	Description
1. Scalable	No loss of quality on zoom.
2. Editable with Code	You can create and control it with HTML, CSS, and JavaScript.
3. Small File Size	Ideal for logos, icons, and simple graphics.
4. Interactive	Supports animation, styling, and events.
5. SEO Friendly	Text in SVG is selectable and searchable.
