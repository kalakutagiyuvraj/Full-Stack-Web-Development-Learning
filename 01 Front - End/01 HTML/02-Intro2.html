HTML : History and Information

        
HTML : Hyper Text Markup Language

        1. Invented By Tim Berners Lee in 1991
        2. HTML is the Language used to create pages using tags.
        3. It is used to structure the web page using tags.
        4. It is not a programing Language, it is a markup Language.
        5. Markup Language is a Language which is used to sturcture the web page using tags.
        
        
Basic structure

    
        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>

            //Meta content goes here which won't be displayed on the page.
            
        </head>
        <body>

            //Visible content goes here.

        </body>
        </html>
