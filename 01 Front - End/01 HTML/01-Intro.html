How Browser Works ? 

        1. When we type something in Browser, Exa We are asking for www.google.com
        2.Browser sends request to the DNS (Domain Names System) which converts our request into IP adress.
        3. Browser sends HTTP request to server, server search for the result in database.
        4. If something found related request server return it on screen using html, css, js.

HTML : Basic information

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
        </head>
        <body>
            
        </body>
        </html>

        
Let's Understand.

        <!DOCTYPE html> : 1. Is a declaration that tells the Browser to use HTML5.
                          2. Without it Browsermay use old behavior for outdated sites & may not render the site correctly.
                          3. With this browser use modern mod : render the page correctly and consistently.
        <html lang="en"> : 1. It is the root element of the page and all other elements have to be inside this.
                           2. lang is an attribute which tells that to the browser we are using English Language. 
        <head> </head> : 1. All meta datas (The data give info about the data) are written inside this tag.
                         2. This information is not for the user it is for browser Understanding only.
                         3. Example : 
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>Document</title>
        <body> </body> : 1. This is the part where we write visible content section of the web page.
                         2. Everything the user sees on the web page goes here.
