HTML4 Structure

    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
    <html>
    <head>
        <title>HTML4 Page
    </head>
    <body>
        <h1>Welcome
        <p>This is HTML4.

    </body>
    </html>

HTML5 Structure

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>

            //Meta content goes here which won't be displayed on the page.
            
        </head>
        <body>

            //Visible content goes here.

        </body>
        </html>

         Features:
            Simple <!DOCTYPE html>
            Supports semantic elements: <header>, <footer>, <article>, <nav>, etc.
            Native support for audio, video, canvas
            Mobile-friendly and faster loading

Difference


    | Feature                         | HTML4                       | HTML5                                   |
| --------------------------- | --------------------------- | ----------------------------------------------- |
| Doctype Declaration         | Long and complex            | Simple: `<!DOCTYPE html>`                    |
| Semantic Tags               | Not available               | Available (e.g., `<header>`, `<section>`) |
| Multimedia Support          | Needs plugins (Flash, etc.) | Built-in tags: `<audio>`, `<video>`       |
| Form Controls               | Basic input types           | New types like `email`, `date`, `range`         |
| Graphics Support            | Not supported natively      | `<canvas>` and `<svg>` supported          |
| Browser Support for Offline | Not available               | Supports offline caching via Web Storage        |
| Mobile-Friendly             | Not optimized               | Responsive design features                      |
| APIs Support                | No support                  | Supports APIs (Geolocation, Drag & Drop)        |
