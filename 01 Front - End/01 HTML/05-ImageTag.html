Image Tag & Its Attributes

    

    This line uses the HTML <img> tag to display an image on a web page. Here’s a breakdown of each attribute:

    src="images/img1.jpg": Specifies the path to the image file. In this case, it looks for img1.jpg inside the images
    folder relative to the HTML file.
    alt="image1": Provides alternative text that describes the image. This text is shown if the image fails to load and
    is also used by screen readers for accessibility.
    width="300": Sets the displayed width of the image to 300 pixels. The browser will scale the image to this width.

    If the image path is incorrect, the browser will show the alt text instead.
    Setting only the width (without height) keeps the image’s aspect ratio, but if you set both, the image may appear
    stretched.
    Example: This tag will show a 300-pixel wide image called img1.jpg and display "image1" if the image can’t be
    loaded.

    
    
    height="300": Sets the displayed height of the image to 300 pixels. The browser will scale the image to this height.
   

    List of Attributes : 
    1. height: Specifies the height of the image in pixels or percentage.
    2. width: Specifies the width of the image in pixels or percentage.
    3. alt: Provides alternative text for the image, if the image cannot be displayed.
    4. src : Specifies the path to the image file.
    5. title: Provides additional information about the image, displayed as a tooltip when hovered over.
    6. id: unique identifier for the image.
    7. class: Assign the css class to the image, may selects multiple elements.
    8. style: Add inline CSS Styling.
