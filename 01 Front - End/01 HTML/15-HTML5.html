HTML5 stands for HyperText Markup Language version 5.
It is the latest version of HTML, the standard language used to create and structure web pages.
It was developed by W3C (World Wide Web Consortium) and WHATWG to improve the structure, multimedia support, and interactivity of web pages.

4. Graphics and Animation	<canvas> and <svg> used for drawing and animation directly in browser.
5. New Form Controls	Input types like date, email, url, range, etc.
6. Offline Web Apps	Web Storage (localStorage, sessionStorage) allows data to be stored in browser.
7. Geolocation API	Lets websites access user's location.
8. Responsive Design Friendly	Works great with mobile-first design and CSS3.
